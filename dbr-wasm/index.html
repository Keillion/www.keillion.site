About Dynamsoft

Founded in Sep 2003 with the aim of being the dynamic center of software developers, Dynamsoft provides enterprise-class version control software, TWAIN software development kits (SDK) and other document imaging SDKs, with numerous generations for each product. Today many Fortune 500 Companies including HP, IBM, Intel, and Siemens trust Dynamsoft solutions for version control and TWAIN scanning SDK development. Contact Us

Dynamsoft Barcode Reader JavaScript Edition is a JavaScript API for barcode scanning based on the WebAssembly technology. This demo supports scanning barcodes directly from a live video stream.
